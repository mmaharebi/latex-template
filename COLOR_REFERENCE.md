# Quick Color Reference

## Theorem Boxes

| Environment | Color | Use Case | Example |
|------------|-------|----------|---------|
| `theorem` | Royal Blue | Formal mathematical statements | Pythagorean theorem |
| `lemma` | Blue (85%) | Supporting theorems | Helper result |
| `proposition` | Blue (85%) | Formal statements | Properties |
| `corollary` | Blue (85%) | Consequences of theorems | Direct results |
| `definition` | Teal | Foundational concepts | Define terms |
| `example` | Slate Gray | Illustrations | Worked examples |
| `remark` | Purple | Observations | Side notes |
| `note` | Purple (85%) | Additional observations | Comments |

## Utility Boxes

| Environment | Color | Use Case |
|------------|-------|----------|

| `importantbox` | Orange | Key information |
| `questionbox` | Amber | Questions/problems |
| `answerbox` | Green | Short answers |

## Color Values (RGB)

- **primary-dark**: (25, 42, 86) - Deep Navy
- **primary-main**: (41, 98, 255) - Royal Blue  
- **primary-light**: (100, 149, 237) - Cornflower Blue

## Typography

- **Section headings**: Deep Navy, underlined
- **Subsection headings**: Royal Blue
- **Subsubsection headings**: Teal
- **Body text**: Almost black
- **Links**: Royal Blue
- **Citations**: Teal

## Box Styling

All boxes use consistent parameters:
- Border width: 0.8pt
- Corner radius: 3pt  
- Padding: 8pt all sides
- Background: 5% tint of frame color
- Title: White text on colored background
